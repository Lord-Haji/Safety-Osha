Wisp Bot's command list:

  • Home
  • Help
  • Data

NOTE: All command require Voice or above in the current room, unless specified otherwise. Commands that require Voice will still work in PM for regular users.

Administrative

console: Sends you a (temporary) link to the console output of the bot. Requires either global admin or for you to be the owner of the bot.

leave: Makes the bot leave, and take the room off of autojoin (if autojoin is provided by inviting the bot). If the bot is set to autojoin by Config, the bot will leave, but rejoin next restart. Requires Room Owner in the room.

Help

help (topic): Links to the bot's documentation about a certain topic. For a list of topics, use .help without a topic specified.

git: Returns a link to the bot's Github repository.

Last.fm

lastfm (username): Displays what the given last.fm username is currently listening to or has last listened to, and provides a link to their profile. If no username is provided, it will attempt to use your registered info.

registerlastfm (username): Works only in PM. This will register the given last.fm username to your PS username, and allows you to use simple .lastfm to link to your last.fm page. To update information, simply use the command again with the updated last.fm username.

Moderation

moderation [on/off]: Turns moderation in this room on or off. Requires Room Owner in the room.

autoban (user): Adds a user to the room blacklist. Requires Room Leader in the room.

unautoban (user): Removes a user from the room blacklist. Requires Room Leader in the room.

blacklist: A list of all the autobanned users of the room.

Quotes

quote (quote): Adds the given quote to the quote page for this room.

quotes: Provides a link to the quote page of the current room.

gquote (id): Says a specified quote.

dquote (id): Deletes a specified quote. Requires either global admin or for you to be the owner of the bot.

Games

kunc: Start a new kunc game.

kunc end: Stops the game of kunc.

kg (answer): Guesses the answer for a game of kunc.

trivia: Starts a new trivia game.

trivia end: Stops the game of trivia.

trivia list: Gives a list of all trivia questions. Requires global admin or bot owner.

tg (answer): Guesses the answer for a game of trivia.

User Info

rank (user): Gives the ratings of the specified user on the smogon ladder.

Custom Commands

addcom (commandName, message): Adds a custom command for that room only. Requires Leader.

delcom (commandName): Deletes a custom command. Requires Leader.

comlist: Gives a list of all the custom commands for that room.

Tournaments

runtour (time): Starts the tour in x minutes, with autodq on 2 minutes and a tour remind. More tour configuration coming soon!
